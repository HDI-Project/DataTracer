solve(tables, primary_keys=None)[source]¶

Solve the foreign key detection problem.

The output is a list of foreign key specifications, in order from the most likely to the least likely.

Parameters

  • tables (dict) – Dict containing table names as input and pandas.DataFrames as values.

  • primary_keys (dict) – (Ignored). This particular implementation does not use this argument.

Returns

List of foreign key specifications, sorted by likelyhood.

dict
